## 适合谁，不适合谁

**适合这些场景：**

- 每天晚高峰需要稳定低延迟的用户（游戏、视频会议、直播推流）
- 在国内跑需要回程优化的业务服务器
- 对 CN2 GIA 或软银线路有明确需求的开发者
- 愿意为稳定性多付钱、不想频繁换机器的人

**不适合这些场景：**

- 月预算低于 $10且对延迟没有要求
- 只是偶尔测试用，不需要长期稳定线路
- 需要大量存储或超高流量（DMIT 流量配额相对保守）
- 需要 Windows 系统（DMIT 主要提供 Linux 环境）

## 三个常见误解，直接说清楚

**误解一：DMIT 贵是因为在割韭菜**

不是。CN2 GIA 带宽成本本身就高，DMIT 的定价和同类高端线路服务商（如搬瓦工 CN2 GIA-E 旗舰套餐）相比并不离谱，部分套餐甚至更便宜。贵是线路成本决定的，不是品牌溢价。

**误解二：香港节点延迟一定比日本低**

不一定。物理距离近不等于延迟低。日本软银线路对华南用户的实测延迟有时比香港 T1 线路更稳定。选节点要看你的出口运营商和目标用途，不能只看地图距离。

**误解三：年付一定比月付划算**

要看套餐。DMIT 部分套餐年付折扣力度在 15%–20% 之间，但热门套餐库存紧张，先月付试用一两个月再决定是否年付，是更稳的策略。

## FAQ

**Q：DMIT 的套餐会随时涨价吗？**

老用户续费价格锁定，不会因为官网调价而影响已有套餐。新购套餐以下单时的价格为准。

**Q：香港 T1 和 Pro 的区别是什么？**

T1 走 CN2 GIA 标准线路，Pro 走 CN2 GIA Premium 线路，后者在高峰期的优先级更高、丢包率更低。价格差约 4 倍，适合对稳定性要求极高的用户。

**Q：流量超出套餐上限怎么办？**

超出后带宽会被限速，不会额外扣费。可以选择购买额外流量包或等下个月重置。

**Q：DMIT 支持 IPv6 吗？**

大部分套餐提供 IPv4 + IPv6 双栈，具体以套餐详情页为准。

**Q：可以用来建站吗？**

可以。Linux 环境，支持常见的 LAMP/LEMP 栈，没有限制建站用途。香港节点对国内访客的加载速度有明显优势。

**Q：新用户有优惠吗？**

DMIT 不定期推出促销套餐，通常在节假日或新节点上线时出现。直接通过官网注册是获取最新优惠的唯一可靠途径。
